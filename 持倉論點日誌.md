# 持倉論點日誌（Thesis & Position Log）

> 四層框架的第三、四層。**進場前必須把每一欄填完**——逼自己想清楚「為什麼買、為什麼會變好、什麼時候我錯了、賠率夠不夠」。
> 這是整套系統裡最少人做、但最值錢的一層（Howard Marks：早=常常先錯，紀律決定期望值）。

## 欄位說明
| 欄位 | 填什麼 |
|---|---|
| 標的 | 代號 名稱 |
| 進場日 / 價 | 日期、成本價 |
| 分類 | ⭐A級買點 / ✅B級買點 / 🔄循環(看PBR) / 🌱拐點 / 🔥強拐點 |
| **L2 訊號** | 拐點掃描亮幾個(A毛利拐頭/B月營收/C EPS加速/D ROE回升)、體檢評等 |
| **L3 催化劑** | 為什麼「會變好且持久」？新產品/客戶/換CEO/產業結構/生活觀察。**要能解釋 L2 為何持續** |
| 量 vs 價 / 結構 vs 循環 | 毛利改善是漲價(好)還是衝量(普)？結構性(持久)還是循環(會回吐)？ |
| **L4 證偽條件** | 「具體數字＋時間＋不可協商」的離場：例「下兩季毛利沒回升>1.5pp、或月營收連3月YoY轉負 → 出場」。**加時間停損**：N季沒兌現也出 |
| 部位 % | 確信高=大、不確定=小 |
| 目標價 / 停損 / 賠率 | 上漲目標、停損價，賠率=(目標-現價)/(現價-停損) **需 ≥2** |
| 檢視日 | 下次檢查論點的日期(通常下次財報/月營收) |
| 現況 | 論點兌現中 / 待觀察 / 已證偽(出場) |

## 持倉

### 範例（請依此格式填，刪掉本列）
- **標的**：2753 八方雲集
- **進場日/價**：2026-06-23 / 187
- **分類**：✅B級買點 + 🌱初拐點
- **L2 訊號**：拐點 2/4（B月營收+13%✓、D ROE回升✓；A毛利待補、C EPS加速✗）；體檢 A 級、含金量 1.93
- **L3 催化劑**：梁社漢排骨等新品牌展店放量 + 本業提價；高現金特許加盟模式
- **量/價・結構/循環**：價(提價)為主、結構性（餐飲剛需、非循環）→ 偏持久
- **L4 證偽**：下兩季 EPS 沒回升、或月營收連 3 月 YoY 轉負 → 出場；時間停損：3 季沒兌現
- **部位**：3%（B級偏防禦，部位中等）
- **目標/停損/賠率**：目標 230 / 停損 165 → 賠率 (230-187)/(187-165)=1.95 ≈ 2 ✓
- **檢視日**：下次月營收公布（每月10日）
- **現況**：待觀察
