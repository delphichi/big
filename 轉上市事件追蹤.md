# 轉上市事件追蹤（Listing-Transfer Event Tracker）

> **為什麼追這個**:轉上市 = 兩種確定性 alpha。

>       + 新聞/公開資訊觀測站的「確定時程」事件(人工補,比預測更確定)。

## 台灣轉上市門檻(2026,新聞整理)
企業從上櫃/興櫃「轉板」至一般上市板,通常需滿足:
| 條件 | 標準 |
|---|---|
| 設立年限 | 滿 3 年以上 |
| 實收資本額 | 達新台幣 6 億元以上 |
| 記名股東人數 | 達 1,000 人以上(內部人持股比例符合規定) |
| 獲利能力 | 最近年度 + 最近會計年度符合獲利標準(稅前淨利佔比達門檻) |

> 用途:未來可用這幾條去篩「即將符合轉上市資格」的大型興櫃/上櫃股,提前布局。

## 三、系統盲區與待辦

- ❌ **興櫃資料未接**:(B) 類型(台鎔/床的世界/儒億)無法自動偵測,只能靠新聞人工補。
  - 若要 cover:需接興櫃市值/財報源(FinMind 興櫃資料有限),或定期掃公開資訊觀測站「上市櫃審議」公告。
- ⚠️ **大型轉上市候選多為循環股**:信驊/群聯/台燿/環球晶/世界/穩懋… forward PE 豁免,看 PBR 不看 PE。
  「有被動買盤 ≠ 該追」——真正「便宜+好公司」的只有 **元太(8069)**。
- 📌 **確定時程 > 市值預測**:第一區(新聞事件)有日期,第二區(市值預測)只是「資格達標」。
  有日期的才好抓進場時點。
